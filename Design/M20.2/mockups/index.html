Shared system summary

Type: Newsreader (serif headlines, ledger feel) · IBM Plex Sans (UI) · IBM Plex Mono (numbers, statuses). All swappable for a generic sans per brief §2.

Palette: warm vellum text on near-black panels. No saturated chroma. Accents are muted "ink" colors.

Severity ramp: one four-step ramp stable → strained → dangerous → critical reused everywhere a "this is getting worse" signal is needed.

Per-act identity: a data row of { numeral, theme word, accent hex }. Acts 3–5 ship as three more rows; no relayout.

What this isn't

Not a rebuild. Combat keeps its structure. No new states, no new mechanics, no map / equipment / meta-progression / leaderboards.

Not painted art. All theme is conveyed with color + label + glyph + layout. Placeholders use striped fills and bracketed text.

Not motion. Color flashes and state changes only. The two background gradients in End / Reward fall back to a flat fill identically.

Not finished copy. Encounter names, danger lines, and the demonic-theme statements are example fills; final copy comes from the game-data layer.

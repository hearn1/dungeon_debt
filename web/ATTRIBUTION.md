# Third-party art attribution

Dungeon Debt ships with art sourced from [game-icons.net](https://game-icons.net) under the **Creative Commons Attribution 3.0 Unported (CC BY 3.0)** license.

License text: <https://creativecommons.org/licenses/by/3.0/>

License floor for this project (locked R005-1): **CC0 and CC BY** only. CC BY-SA (copyleft) and any Non-Commercial (NC) variant are explicitly rejected so the game stays free to ship as a paid commercial release.

Icons may be modified (recolored / cropped / composited) under CC BY 3.0 §3(a). All icons in `web/assets/` are used unmodified except for color (white foreground on transparent background, applied via the game-icons.net URL color-parameter endpoint).
